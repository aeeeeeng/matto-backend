## INSTALL

Install vendor package
<pre>
    <code>
        composer update
    </code>
</pre>

Generate all provider
<pre>
    <code>
        php artisan vendor:publish
    </code>
</pre>

Generate API Documentation
<pre>
    <code>
        apidoc -i apidoc/ -o public/doc/ 
    </code>
</pre>

you can access the doucmentatin in route http://yourdomain/docs/
